PACE

Parallel Association Chain Evaluation. For each seed word, generate 3 parallel 20-word association chains. Score = position-weighted average semantic distance (FastText crawl-300d-2M) across positions and chains using weights 1/[(n-1)(i-1)] for the (i-1)→j gap.
