Submit a metric

Contributions are welcome. Submission is by pull request, modeled on the ARC-AGI Community Leaderboard.

  1. 1. Fork the repository
    github.com/jean-technologies/creative-ai-index
  2. 2. Copy the template directory
    cp -r submissions/.example submissions/my-metric-id
    Use a short, lowercase, hyphenated id (e.g. forward-flow-v2).
  3. 3. Fill in submission.yaml
    See submissions/.example/submission.yaml for the full schema. Required: name, authors, description, code_url, and one entry in versions.
  4. 4. Provide scores.json
    JSON object mapping canonical model_key to a single float score. Use the model_keys from data/canonical_models.json. Coverage of all 68 models is not required — partial coverage simply produces fewer rows for that metric on the leaderboard.
    {
  "anthropic_claude-opus-4-6": 89.7,
  "openai_gpt-5-4": 88.3,
  "google_gemini-2-5-pro": 87.2
}
  5. 5. Open a pull request
    A maintainer will review for schema validity, sanity-check the scoring code at code_url, and merge. The site rebuilds automatically on merge.

Verification policy

Scores are self-reported and have not been independently verified. Maintainers may verify submissions at their discretion. The required code_urlfield exists so the community can audit any submission’s scoring procedure.

Updating an existing metric

Add a new entry to the versions array. Do not edit prior versions — the version history is the audit trail.
